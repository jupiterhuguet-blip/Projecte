Mini arcade

Aquesta aplicacio ens permetra jugar al joc pedra, paper o tisora, podrem triar les rondes que prefereixim jugar, i despres podrem jugar al joc de endevinar el numero, per poder acertarlo més facilment ens anira dient si el numero que em dit esta per sota o es més alt que el numero per endevinar.

Per fer-lo funcionar sols aurem de iniciar-lo i aurem de donarli si 1 o 2 sengons el joc que vulguesim
